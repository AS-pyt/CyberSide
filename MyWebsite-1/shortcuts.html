Windows Shortcut Keys

Boost your productivity with these keyboard shortcuts:

🛠️ Basic

  • Ctrl + C – Copy
  • Ctrl + V – Paste
  • Ctrl + X – Cut
  • Ctrl + Z – Undo
  • Ctrl + A – Select All
  • Alt + Tab – Switch between apps
  • Win + D – Show desktop

📁 File Explorer

  • Win + E – Open File Explorer
  • Alt + Enter – Properties
  • F2 – Rename

🌐 Browsers

  • Ctrl + T – New Tab
  • Ctrl + W – Close Tab
  • Ctrl + Shift + T – Reopen Last Closed Tab
